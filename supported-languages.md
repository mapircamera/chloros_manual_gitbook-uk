## Як змінити мову

### У Chloros Desktop/Browser

1. Відкрийте налаштування програми.
2. Перейдіть до меню вибору мови.
3. Виберіть бажану мову зі списку.
4. Інтерфейс оновиться миттєво.

### У Chloros CLI

Використовуйте команду `language`, щоб переглянути або змінити мову інтерфейсу CLI:

```bash
# View current language
chloros-cli language

# Change to Spanish
chloros-cli language es

# Change to Chinese (Simplified)
chloros-cli language zh

# Change to Brazilian Portuguese
chloros-cli language pt-BR

# List all available languages
chloros-cli language --list
```

Для отримання додаткової інформації див. [документацію CLI](CLI.md).

### У Chloros Python SDK

Встановіть параметр мови під час ініціалізації SDK, щоб отримувати повідомлення та результати на бажаній мові.

## Покриття

Усі 38 мов повністю підтримуються в:

* **Chloros Desktop** - повний переклад графічного інтерфейсу користувача
* **Chloros Browser** - веб-інтерфейс на всіх мовах
* **Chloros CLI** - інтерфейс командного рядка та вихідні повідомлення
* **Chloros Python SDK** - API повідомлення та документація

Підтримка мов забезпечує користувачам у всьому світі можливість ефективно працювати рідною мовою без бар&#x27;єрів.
